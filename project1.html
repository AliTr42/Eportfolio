Project Multidisplinary 1

In Multidisciplinary Project 1, my team and I had to carry out a project, that included knowledge like electronics and C language programming. Our main objective was to create a game using two dice. Me and my teammates decided to choose a game in which every person who played threw both dice and added up the numbers. If they exceeded 55, the person would lose. therefore, regarding the electronics part, we had to use 12 LEDs to display the dice numbers. What about the programming part, we had to write in a code that suited the game we created. Finally, we needed to connect the electronics part and the programming part but unfortunately, nobody had the time to do that.
